背景
lenny-railsからファイルをコピーして、使える部分は流用し、とりあえずDB　shogi84にデータを登録したい
lenny-railsではディレクトリを分けずに１箇所にいろいろなツールを置きすぎ、文書もろくに残さないので
後から見てなんのためのファイルなのか、必要なのかすら不明になってしまったという反省のもとにこのディレクトリをつくった
結果としては2008年以前のファイルはほとんど捨て、2009年につくった2chデータベースの内容を登録するスクリプトを使うことになった。

このディレクトリの目的
shogi84に,24形式の棋譜データを登録するためのツールの置き場所である。

各論
 ../lib/db_connect.rb

   (pg_hba.confでmd5とする。）
